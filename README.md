# Sample for Webview

Run

```
# Linux
$ go build -o webview-linux && ./webview-linux

# MacOS
$ go build -o webview-mac && ./webview-mac

# Windows
$ go build -ldflags="-H windowsgui" -o webview-windows.exe
```

Package MacOS App Example

```
make
```
